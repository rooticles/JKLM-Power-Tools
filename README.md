# 🚀 JKLM Spacebar to Hyphen & Helper Tool
A powerful Tampermonkey userscript for JKLM.fun (especially Bombparty), designed to enhance your typing experience, assist with word finding, and track your stats.

## 🛠 Features
### 1. ⌨️ Keyboard Optimization (🐱)
- Spacebar to Hyphen : Automatically converts the spacebar into a hyphen ( - ) when it's your turn in the game, saving you time when typing compound words.
- Chat Mode : This feature can optionally be enabled for the chat as well.
### 2. 📚 Intelligent Dictionary (📚)
- Search : Find words based on the current syllable.
- Click-to-Copy : Simply click on a suggested word to instantly copy it to your clipboard.
- Used Words Tracker : The script automatically remembers which words have already been used in the current round and filters them out of the suggestions.
- Special Filters :
  - /s : Short words (2-5 letters)
  - /l : Long words (20-30 letters)
  - /h : Words with a hyphen
  - /r : Rare words (containing X, Q, Y, etc.)
### 3. 📊 Statistics Dashboard (📊)
- Tracks your total words submitted.
- Saves your longest word .
- Calculates your average response time (from the start of your turn until your word is submitted).
### 4. 🛡 Admin Panel & Visuals
- Chat Macros : Buttons for quick responses (GG, WP, GL HF).
- Theme Color : Customize the script's highlight color.
- Syllable Highlighter : (Requires admin login) Visually emphasizes the current syllable in the game.
## 📥 Installation
1. Install the Tampermonkey browser extension.
2. Create a new script in the Tampermonkey dashboard.
3. Copy the code from the jklm_spacebar_to_hyphen.user.js file into the editor.
4. Save the script ( Ctrl + S ).
5. Open JKLM.fun and join a game of Bombparty.
## 🔐 Unlocking the Admin Panel
To unlock the Admin Panel (🛡) and its visual aids, you need to log in within the script:

1. Click the 🛡 icon .
2. Enter one of the authorized usernames and passwords (these are hardcoded in the script).
3. After logging in, you will have access to UI color customization and macros.
